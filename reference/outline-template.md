# 大纲模板（可按类型裁剪）

## 通用框架
1. 摘要（目标/读者/结论/行动项）
2. 背景与目标（问题/机会、范围、不做什么）
3. 关键约束/假设
4. 主要内容（按类型展开）
5. 风险与缓解
6. 里程碑与负责人
7. 参考/附录

## 技术方案侧重
- 约束与需求摘要
- 方案选项与对比（表格：选项/优点/缺点/适用前提）
- 决策准则与推荐方案
- 技术架构/流程/接口要点
- 交付与里程碑，运维与监控

## 产品设计侧重
- 用户与场景（旅程、痛点/机会）
- 目标与成功度量（可量化指标）
- 方案概述（信息架构、核心流、关键交互）
- 范围与优先级（MVP / V1+）
- 验证计划（原型测试、实验、指标）

## 战略规划侧重
- 目标与北极星指标
- 外部态势与趋势（机会/威胁）
- 战略选项与取舍（路径比较）
- 战略路径与阶段性里程碑
- 资源配置与风险管理
